# Beacon

**Beacon** 是一款极简、无干扰的 Chrome 浏览器插件，专为深度阅读、资料搜集和长文研究设计。它允许用户在任何网页上为选中的文本添加高亮标记，并在页面右侧边缘生成位置对应的颜色模块。通过点击这些颜色模块，用户可以瞬间精准定位回原文位置。

## 🎯 产品定位

**“你的专属资料采集与核心信息标记工具。”**

在浩如烟海的网页、长篇文档或技术资料中，我们常常需要标记和摘录最重要的内容。Beacon 专注于**高价值资料的锚定与追踪**，创新性地将高亮映射到浏览器视窗的**右侧边缘**。
它不破坏网页原有的排版，在不干扰正常阅读的前提下，为你提供了一个全局视角的“内容骨架”。无论你是需要重点关注某段关键数据，还是整理文章的核心脉络，只需在页面边缘轻轻一点，即可穿梭于需要反复查阅的关键信息之间。

## ✨ 核心功能

*   **⚡ 快捷标注:** 选中文本后，使用系统快捷键（Mac 默认 `Command + Shift + F`，Windows 默认 `Alt + Shift + F`）立即生成标记，无需打断阅读心流。**（提示：您随时可以在 Chrome 扩展管理界面的“键盘快捷键”选项中，根据个人习惯自定义修改此键位）**。
*   **📍 智能排序与原生级跳转:** 侧边颜色块严格按照文本在页面中的实际出现顺序排列。点击色块，底层调用浏览器原生搜索定位能力，无视复杂的嵌套滚动条，**瞬间平滑滚动并绝对聚焦**到原文本位置。
*   **👻 沉浸式与无干扰设计:** 
    *   **静态低调:** 闲置状态下，高亮块呈现低饱和度的莫兰迪色系，完美融入页面边缘，绝不喧宾夺主。
    *   **动态放大:** 当鼠标悬停或交互时，模块会平滑放大，提供清晰的视觉反馈和更大的点击热区。
    *   **悬停预览:** 鼠标悬停在色块上方，即可通过浮窗展现标注的原出处文本。
*   **💾 本地持久化记忆:** 所有的标注数据基于当前网页的 URL (域名+路径) 自动在本地存储 (`Local Storage`)。刷新页面或下次重新打开，你的“阅读痕迹”依然历历在目。
*   **🎛️ 便捷的管理侧边栏:** 点击插件图标即可呼出侧边栏，支持查看当前快捷键，并提供带二次确认的安全“一键清除当前页面高亮”功能。

## 🛠️ 使用场景

*   **学术/长文阅读:** 在数万字的文章中，一边阅读一边使用快捷键标记论点，读完后通过右侧色块快速回顾整文脉络。
*   **开发文档查阅:** 在长达几十页的 API Reference 中，标记你需要频繁跳回查看的几个关键接口参数。
*   **资料搜集与对比:** 在维基百科或长篇财报中，标记互相关联的几段数据，通过点击色块在上下文之间快速切换。

## 🚀 安装说明 (开发者/本地加载模式)

1. 克隆或下载本仓库到本地计算机。
2. 打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/` 并回车。
3. 在页面右上角开启 **“开发者模式” (Developer mode)**。
4. 点击左上角的 **“加载已解压的扩展程序” (Load unpacked)**。
5. 选择本仓库所在的文件夹 (`Highlighter`)。
6. 安装成功！建议将插件图标固定在浏览器工具栏。

## 📝 架构与实现细节

- **非侵入式 DOM 渲染:** 与传统的直接修改和包裹网页 DOM 树的高亮插件不同，我们采用独立的浮层渲染。避免了破坏复杂网页（如 React/Vue 单页应用）的内部状态。
- **稳健的文本寻址:** 废弃了脆弱的 DOM 节点计算，转而依赖文本内容的相对位置进行排序，并采用浏览器原生的 `window.find` 配合 `.scrollIntoView` API 进行绝对跳转，兼顾了性能和各种异构页面的兼容性。
